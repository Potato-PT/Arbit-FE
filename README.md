# Arbit UI

React, TypeScript, Vite 기반 전시 추천 UI 프로토타입입니다.

## Scripts

```bash
npm run dev
npm run build
npm run lint
```

## Routes

- `/` 홈
- `/exhibitions/search` 전시 검색
- `/exhibitions/:id` 전시 상세
- `/exhibitions/:id/review` 후기 작성
- `/user/login` 로그인
- `/user/signup` 회원가입
- `/user/preferences` 취향 설정
- `/user/mypage` 마이페이지
